# あみだくじ抽選

ブラウザだけで動く、あみだくじ風の抽選Webアプリ。

## 使い方

`https://little-hands.github.io/amidakuji/` にアクセス。

1. 候補名を改行区切りで入力
2. 当選数を指定
3. **START** ボタンを押すと、上のカードがフリップして「当たり / ハズレ」が見える
4. 当たりカードから玉が出てあみだの線をたどり、下端の名前に到達 → 当選

`Cmd / Ctrl + Enter` でも抽選実行可。

## 構成

- `index.html` — マークアップ
- `style.css` — スタイル
- `app.js` — 抽選ロジック（SVG描画 + アニメーション）

外部依存なし（CDN・フレームワーク不要）。
